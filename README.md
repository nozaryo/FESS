# FESS(F.T.E. Rocket Simulating System)
[F.T.E.](https://www.fte-tohoku.org/)内で使用しているロケットフライトシュミレータの開発リポジトリ.  
[Shun0228氏のRocketSimulation](https://github.com/Shun0228/RocketSimulation)をもとに開発しているものです.
## Features
下記のような機能があります.
~~~
F.E.S.S.  
├─Detail mode  
│  ├─Trajectory  
│  └─Open Parachute  
└─Scatter mode  
    ├─Trajectory  
    │  ├─Single output  
    │  └─Multiple output  
    └─Open Parachute  
        ├─Single output  
        └─Multiple output  
~~~
