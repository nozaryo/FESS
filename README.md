# FESS(F.T.E. Rocket Simulating System)
[F.T.E.](https://www.fte-tohoku.org/)内で使用しているロッケトフライトシュミレータの開発リポジトリ.  
[Shun0228氏のRocketSimulation](https://github.com/Shun0228/RocketSimulation)を参考に開発しているものです.
## Features
~~~
F.E.S.S.  
├─Detail mode  
│  ├─Trajectory  
│  └─Open Parachute  
└─Scatter mode  
    ├─Trajectory  
    │  ├─Single output  
    │  └─Multiple output  
    └─Open Parachute  
        ├─Single output  
        └─Multiple output  
~~~
